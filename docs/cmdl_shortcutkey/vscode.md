# vscode

**ctrl + K, V : 侧边栏预览**

`ctrl + ]/[` 回退或前进

`ctrl + enter` 向下回车

`ctrl + shift + k` 删除一行

`ctrl + shift + ]/[` 折叠该区域

`F12` 进入定义，`ctrl + alt + -` 导航回退

选中一段code之后，使用 `ctrl + D` 一个一个选中相同的code。或者 `ctrl + shift + F` 全选。

`ctrl + P` + `@:` 转到文件中的符号

`ctrl + home` 到文件头

`ctrl + alt + ↓` 多光标选择，将光标设置在当前位置的上方或下方
