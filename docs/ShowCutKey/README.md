# VS Code快捷键

**ctrl + K, V : 侧边栏预览**

ctrl + ]/[ : 回退或前进

ctrl + enter: 向下回车

ctrl + shift + k : 删除一行

ctrl + shift + ]/[ : 折叠该区域

F12 : 查看声明

ctrl + alt + - : 返回
